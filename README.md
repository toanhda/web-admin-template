# Template web admin

## Overview

This is template can help you build project admin so fast

## Architecture

### Front-end

ReactJs

### Back-end

 We have 2 option:
 - Java
 - TypeJS
 
 ## Installation
 
 ### Front-end

#### Requirements

- Nodejs
- NPM

#### Start

```shell
cd client
npm install
npm start
```

### Back-end

#### Java

##### Requirements

- Java 8


##### Build

```shell
cd server/java
mvn clean install
```

#### TypeJS

TDB
